
# IPv4 Addressing

Configure IPv4 addresses on router interfaces and end hosts, verify with show commands, and test connectivity with ping.

## Objectives

- Set the hostname on R1
- Assign IP addresses to R1's interfaces and enable them
- Add interface descriptions
- Verify configuration with show commands and save the running config
- Configure IP addresses on PC1, PC2, and PC3
- Test connectivity by pinging between all three subnets

## Network Topology

Three subnets connected through R1:

- 15.0.0.0/8 — PC1 and SW1
- 182.98.0.0/16 — PC2 and SW2
- 201.191.20.0/24 — PC3 and SW3

## Tools

- Cisco Packet Tracer
